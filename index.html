HYPER TEXT MARKUP LANGUAGE (HTML)

HTML stands for Hyper Text Markup Language. It is one of the most famous and widely used markup language, especially for web development. HTML act as the founding stone of any web page along with other web technologies like CSS, JavaScript. One can not neglect importance of HTML. There are uncountable uses of HTML. One of the main use of HTML is to structure the web pages. It is very easy and very useful computer language. Probably Most of the software engineer from any domain started their computer career from coding in HTML.
Developers divides web development into two main classes:

  • Front-end
  • Back-end

More frankly HTML is considered as front-end technology. HTML language consists of the number of tags each tag has the opening tag and ending tag except some exceptional cases. Opening tag starts like <open> this and ends like </close> this.
HTML is the most important, common and easy to use computer language. It belongs to markup language category. HTML is to web development, what water is to life. Uses and importance of HTML is abundant. HTML occupies the big number of internet’s volume. It is the foundation to the web development. Without HTML your browser cannot be able to display you a website.

SYNTAX:

The syntax of HTML is very simple to understand. Every content is represented by a tag. 

 
<html>
	<body>
		<p> <strong> This is a paragraph </strong> </p>
	</body>
</html>

OUTPUT:

This is a paragraph

<p> tag defines text as a paragraph and <strong> tag make the text strong.
